# mandrex
Financial evaluator

---

## How to Use This Repository

1. Create a folder named `mandrex`.
2. Copy all the file contents above into their respective files (including the `examples/` subfolder with a dummy Excel file).
3. Install the required packages.
4. Place your own financial Excel file (or use the sample one) and run `python main.py`.

The tool is fully functional and can be extended with additional ratios, forecasting, or interactive dashboards.

---

If you need further enhancements (e.g., support for multiple sheets, more sophisticated analysis, or a web interface), feel free to ask!
